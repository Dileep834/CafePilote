# CafePilot ERP - Project Setup

## Goal

Build a scalable ERP/POS for cafes and restaurants.

## Stack

-   React 19 + Vite + TypeScript
-   TailwindCSS + shadcn/ui
-   Zustand
-   TanStack Query
-   React Router
-   Axios
-   Framer Motion

## Principles

-   API-first
-   Modular
-   Multi-company
-   Mobile, Tablet, Desktop responsive
-   Production-ready
