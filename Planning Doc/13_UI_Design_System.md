# UI Design System & Component Library

## 1. Design Tokens & Theme
- **Color Palette**: Modern dark mode & high-contrast POS theme.
  - Primary Accent: Amber / Warm Gold (`#F59E0B`)
  - Secondary: Emerald Green for Success / Paid (`#10B981`)
  - Background: Deep Slate (`#0F172A`)
  - Surface Card: Slate Gray (`#1E293B`)
- **Typography**: Inter / Outfit sans-serif fonts for clear readability on POS touchscreens.

## 2. Component Guidelines
- **Touch Targets**: Minimum `48px x 48px` for all buttons to support finger touch interaction.
- **Cart List**: Fixed sidebar with sticky order summary footer.
- **Category Bar**: Horizontal swipeable scrollbar with quick filter buttons.
