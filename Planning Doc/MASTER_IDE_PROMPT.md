# CafePilot ERP Master IDE Prompt

Build CafePilot ERP as a production-ready, modular ERP system.

Requirements: - React 19 + Vite + TypeScript - TailwindCSS + shadcn/ui -
Responsive (mobile, tablet, desktop) - Detachable modules -
Multi-company, multi-outlet - RBAC - API-first architecture - Clean
Architecture + SOLID - Lazy loading - Zustand + TanStack Query -
Reusable UI components - Dashboard, POS, Inventory, Purchase, Kitchen,
CRM, Reports, Franchise, Settings

Implement each module independently under `src/modules/`. Each module
owns its routes, services, store, components, pages, hooks, and API. No
cross-module coupling.
